### Added

- Initial monorepo scaffold with Nx workspace
- Team-based directory structure (platform, orders, products, web, mobile)
- Shared libraries: `@shared/types`, `@shared/utils`
- Domain libraries: `@platform/core`, `@orders/domain`, `@products/domain`
- Architecture documentation (`ARCHITECTURE.md`)
- Onboarding guides for backend, frontend, mobile, and DevOps engineers
- Governance framework (ARB, RFC process, dependency policy)
- CI pipeline with GitHub Actions and Nx Cloud
- EditorConfig and Prettier for multi-stack formatting
- AI agent configurations (Cline, Cursor, Claude Code)
